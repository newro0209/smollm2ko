# datasets

이 폴더는 데이터셋 원본, 가공 결과, 테스트 프롬프트를 일관된 구조로 관리하기 위한 영역입니다.

## 구조

- raw: 포맷이 제각각인 데이터셋 원본을 보관합니다.
- processed: 일관된 스키마로 정제된 최종 데이터셋을 보관합니다.
- prompts: 테스트 및 평가에 사용하는 프롬프트를 보관합니다.

## 규칙

- 각 데이터셋은 폴더 단위로 분리합니다.
- 데이터셋 폴더에는 출처, 라이선스, 요약 설명을 함께 기록합니다.
- 원본은 절대 수정하지 않고, 변경이 필요하면 processed로 분리합니다.
